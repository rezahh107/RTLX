# Performance acceptance evidence

Run 30 cold measurements in the pinned release environment for small (100), medium (1000), and large (5000) eligible-text-node fixtures. Record discovery, classification, tokenization, write, observer callback, queue depth, memory trend, rollback, long tasks, and attributable layout shift. The budgets in `registries/performance-budgets.v2.json` are acceptance targets, not measured results in this repository snapshot.
