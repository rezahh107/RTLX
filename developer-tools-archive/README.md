# RTLX Developer Tools Archive

These files preserve advanced/developer functionality removed from the focused 15.6.0 extension:

- options and side-panel interfaces;
- element/failure picker and selector generation;
- profile authoring helpers and related tests;
- the v15.5.5 background entry containing advanced message handlers and context-menu/control-panel wiring.

The build has no entry point or copy step for this directory. These sources are included only in the source/delivery archive and are absent from browser release packages.
