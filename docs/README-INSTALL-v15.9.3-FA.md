# راهنمای نصب RTLX 15.9.3

## Chrome و Edge

1. فایل `RTLX-v15.9.3-UNPACKED-INSTALL.zip` را Extract کنید.
2. در Chrome نشانی `chrome://extensions` و در Edge نشانی `edge://extensions` را باز کنید.
3. نسخهٔ Unpacked قبلی RTLX را حذف یا غیرفعال کنید.
4. Developer mode را فعال کنید.
5. روی **Load unpacked** کلیک کنید.
6. برای Chrome پوشهٔ `LOAD-UNPACKED-CHROMIUM` و برای Edge پوشهٔ `LOAD-UNPACKED-EDGE` را انتخاب کنید.
7. نسخهٔ نمایش‌داده‌شده باید `15.9.3` باشد.
8. صفحهٔ هدف را Reload کنید.

## گرفتن گزارش معتبر

- هنگام فشردن دکمهٔ گزارش، Tab هدف را باز و Visible نگه دارید.
- اگر صفحه هنگام Capture مخفی یا Runtime غیرفعال شود، نسخهٔ 15.9.3 فایل ناقص دانلود نمی‌کند.
- به Tab هدف برگردید و دوباره گزارش بگیرید.

## مرز اعتبارسنجی

عبور تست‌های محلی به معنی اثبات کامل رفتار روی تمام سایت‌های واقعی نیست. برای تأیید نهایی، Artifact همین نسخه باید روی صفحهٔ واقعی و با تغییر Visibility آزمایش شود.
