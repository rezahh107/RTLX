# Migration Guide — RTLX 15.5.1 to 15.5.2

1. Remove the previously loaded unpacked extension from the browser extensions page.
2. Extract `RTLX-v15.5.2-UNPACKED-INSTALL.zip`.
3. In Chrome/Chromium, choose `LOAD-UNPACKED-CHROMIUM` with Load unpacked.
4. In Edge, choose `LOAD-UNPACKED-EDGE` with Load unpacked.
5. Open the target site, click RTLX, choose `همیشه این سایت`, then choose `اصلاح همین تب`.
6. If a site still needs stronger correction, set the site mode to `اجبار فقط روی کانتینرها` from the popup.

No backup migration is required. Existing settings are preserved. Fresh installs default to `auto-safe`.
