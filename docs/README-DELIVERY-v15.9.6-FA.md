# راهنمای بستهٔ تحویل RTLX 15.9.6

این بسته شامل موارد زیر است:

- بستهٔ کامل ادامهٔ توسعه؛
- بستهٔ نصب Unpacked برای Chrome و Edge؛
- بسته‌های Chromium، Edge، Firefox Desktop و Firefox Android؛
- سورس کامل Font-sanitized؛
- Patch نسخهٔ 15.9.5 به 15.9.6؛
- گزارش پیاده‌سازی، Evidence و Manifestهای SHA-256.

این نسخه فقط مشکل تأییدشدهٔ باقی‌ماندن Bullet فهرست فارسی در سمت چپ را اصلاح می‌کند. Text Block اصلی همچنان Paragraph باقی می‌ماند و نزدیک‌ترین `<li>` فقط به‌عنوان مالک مستقل Marker جهت‌دهی می‌شود. ادعای اثربخشی نهایی روی DeepSeek فقط پس از اجرای نسخهٔ 15.9.6 و ثبت Screenshot و گزارش واقعی مجاز است.
