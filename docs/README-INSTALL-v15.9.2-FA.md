# راهنمای نصب RTLX 15.9.2

## Chrome و Microsoft Edge

1. فایل `RTLX-v15.9.2-UNPACKED-INSTALL.zip` را Extract کنید.
2. نسخهٔ قبلی RTLX را در `chrome://extensions` یا `edge://extensions` غیرفعال یا Remove کنید.
3. **Developer mode** را روشن کنید.
4. **Load unpacked** را انتخاب کنید.
5. برای Chrome پوشهٔ `LOAD-UNPACKED-CHROMIUM` و برای Edge پوشهٔ `LOAD-UNPACKED-EDGE` را انتخاب کنید.
6. نسخهٔ نمایش‌داده‌شده باید `15.9.2` باشد.
7. صفحهٔ هدف را Refresh کنید یا در Tab جدید باز کنید.

## گرفتن گزارش معتبر

- Tab هدف باید هنگام گرفتن گزارش Visible باشد.
- اگر افزونه پیام داد که Tab مخفی یا Runtime غیرفعال است، به همان Tab برگردید و دوباره گزارش بگیرید.
- روی بخشی که هنوز مشکل دارد ابزار انتخاب بخش را اجرا کنید تا `selectedElement` در گزارش موجود باشد.
- برای آزمایش Streaming، یک گزارش هنگام تولید پاسخ و یک گزارش پس از پایان و رسیدن Capture به وضعیت `ready` تهیه کنید.

## Firefox

بسته‌های Firefox Desktop و Firefox Android در تحویل وجود دارند. تأیید اجرای Artifact نهایی نیازمند اجرای واقعی روی مرورگر یا دستگاه مربوط است.
