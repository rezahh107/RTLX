# RTLX 15.9.6 Release Notes

RTLX 15.9.6 is a narrowly scoped corrective patch for the confirmed DeepSeek nested-list marker failure.

## Fixed

- Persian and mixed-language text blocks nested inside `<li>` now resolve the nearest list item as a separate marker owner.
- When list repair is enabled and the list item has no explicit `dir`, RTLX applies the same resolved list direction to the marker-owning `<li>`.
- Existing explicit `dir` values on list items are preserved.
- Nested lists use the nearest list-item owner and do not leak direction across list levels.
- Inline code inside an RTL list item remains protected and LTR.
- Marker CSS applies only to RTLX-owned list directions.

## Unchanged

Text-block enumeration, classifier thresholds, site profiles, queue limits, streaming policy, typography selection, permissions, evidence privacy, and explicit host directions remain unchanged.

## Verification status

Local deterministic gates and controlled Chromium smoke are required for release. Real DeepSeek visual confirmation remains an external gate.
