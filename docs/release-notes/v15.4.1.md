# RTLX 15.4.1 — Personal Install Hardening

RTLX 15.4.1 is a small reliability patch for long-term personal use through Developer Mode.

## Added

- Frozen Chromium/Edge public manifest key and deterministic extension ID verification.
- Complete personal backup/export, checksum validation, dry-run import, and bounded restore.
- Atomic local ZIP installer with hash verification, fixed install path, previous-version preservation, and rollback.
- Local health/recovery panel in Options.
- Build-generated package-integrity manifest and startup consistency check.
- Personal build/test/package commands.

## Preserved

- Profile Schema v3 and all frozen language, permission, privacy, font, mutation, rollback, and diagnostics boundaries.
- Separate Chromium, Edge, Firefox Desktop, and Firefox Android packages.
- Distribution-certification tooling remains available but is not part of the ordinary personal workflow.

## Important migration note

The first move from an unkeyed Chromium/Edge unpacked installation to the frozen-key build may change the extension ID. Export state before removing the old unpacked entry. All later 15.4.1+ keyed builds derive the same ID.
