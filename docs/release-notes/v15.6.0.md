# RTLX 15.6.0 — Focused Personal Edition

This release removes advanced/developer surfaces from the shipped extension and concentrates the product around activation, smart repair, font choice, and problem reporting.

Highlights:

- one Persian-first popup with semantic status colors;
- per-site activation and smart reapply;
- bundled Vazirmatn or local-first Persian selection;
- local Amazon Ember Display/Amazon Ember with bundled Inter fallback;
- Persian RTL/right and English LTR/left decisions;
- one-click cleanup of legacy picker rules and site settings;
- user/community profiles ignored by the focused runtime;
- options, side panel, picker, context-menu authoring, and advanced handlers excluded from release builds;
- archived advanced source retained for future development;
- no new permissions, telemetry, upload, remote code, or Amazon font redistribution.
