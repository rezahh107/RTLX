# RTLX 15.8.0 — Structured Text Block Coverage

RTLX 15.8.0 evolves the content pipeline from one direction target per semantic region to deterministic per-block processing for document-like chat responses.

## Added

- ordered Text Block Enumerator;
- independent classification and direction resolution for headings, paragraphs, list items, quotes, definitions, captions, and table cells;
- separate direction and alignment targets;
- logical `text-align: start` for block-capable targets;
- bounded typography continuation beyond the first 50 Text nodes;
- coverage-aware runtime diagnostics;
- deterministic profile category precedence;
- editor-preserve policy across bundled conversational profiles;
- fixtures for structured Markdown, nested blocks, typography continuation, alignment safety, and profile precedence.

## Preserved

- layout/icon safety from 15.7.3;
- permissions and privacy boundary;
- hybrid ownership and deterministic rollback;
- code/math/editor/terminal exclusions;
- local-first Vazirmatn and Amazon Ember behavior;
- no Amazon font redistribution.

## Deferred

CSS Custom Highlight remains an optional future debug renderer. Its absence is not a functional defect in this release.
