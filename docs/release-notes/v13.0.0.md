# RTLX 14.0.0 Release Notes

## Added

- Smart Element Picker with Content, Code, Math, Editor, Terminal, and Ignore categories.
- Deterministic automatic selector generation.
- Profile Builder v2 with import, export, migration, and signing support.
- Selected-element diagnostics panel and active Profile Inspector.
- Offline RTLX profile library for twelve products, including Qwen.
- Offline supported-site detection.
- One-click per-site enable/disable.
- Configurable keyboard command.
- IntersectionObserver candidate admission and integrated FailureManager.

## Security and correctness

- Hardened diagnostic redaction and message authorization.
- Journal-safe rollback and exact mutation ownership checks.
- Complete signed-profile verification sequence.
- Enforced wrapper/token limits.
- Persian-only typography decisions and protected descendant font reversion.
- Firefox minimum version updated to 140 on desktop and 142 on Android, with a repository-stable ID and no-data-collection declaration.

## Known verification gaps

Real-browser E2E, manual screen-reader, pinned performance, store signing/submission, and production remote-profile key/endpoint evidence are not established by source implementation alone.
