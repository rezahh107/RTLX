# RTLX 15.9.5 Release Notes

RTLX 15.9.5 repairs incomplete text-block certification and duplicate content-runtime startup after a background worker restart.

## Fixed

- Discovered unprocessed text blocks are no longer suppressed by generic candidate revision deduplication.
- Capture readiness now includes pending text-block processing and cannot certify incomplete coverage.
- Live stale-epoch content runtimes are rebound instead of reinjected.
- Background document registration is reconstructed without restarting page processing.
- Selected-element evidence is retained when only runtime provenance changes inside the same browser document.
- Report creation no longer reprocesses a healthy page unconditionally.

## Unchanged

Language decisions, site profiles, selector rules, typography behavior, mutation intake, streaming policy, queue limits, permissions, and report privacy remain unchanged.

## Verification status

Local deterministic gates and controlled browser smoke are required for release. Real DeepSeek validation remains an external gate.
