# RTLX 15.5.0 — Failure Evidence Capture

RTLX 15.5.0 adds a local, user-initiated failure report for pages where direction, typography, profile matching, or runtime behavior is wrong.

## Added

- Page eligibility and browser-restriction classification.
- Optional broken-element picker.
- Text-free structural, computed-style, text-shape, profile-rule, runtime, fixture, and diagnostic evidence.
- User-authored expected/actual notes.
- Exact JSON preview before download.
- Schema-locked privacy manifest and canonical JSON output.

## Not added

- No new permission.
- No telemetry or upload.
- No page text, full DOM, cookies, form values, site storage, network logs, console capture, or automatic screenshot.
- No automatic selector repair or remote model integration.
