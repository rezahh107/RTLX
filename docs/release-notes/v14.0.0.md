# RTLX 14.0.0 Release Notes

RTLX 14 focuses on a Persian/English workflow and brings profile editing into a persistent control panel.

## Highlights

- Persian prose is conservatively RTL with Vazirmatn.
- English and technical Latin runs remain LTR with Inter.
- Optional Amazon Ember Display support uses an already-installed local font only.
- Side Panel/sidebar provides live profile inspection and per-selector rule controls.
- Smart Picker presents safe selector candidates instead of requiring CSS knowledge.
- Opt-in editable-field assistance switches Persian input to RTL and English input to LTR.
- Settings may be scoped to a site or privacy-preserving conversation/workspace key.
- Logical list repair addresses nested list markers and indentation.
- Temporary right-click overrides provide fast Content, LTR, and Ignore actions.
- Offline official profiles include Qwen and eleven other supported services.
- A signed community-profile path exists and remains fail-closed until production keys are established.
- Dedicated Chromium, Edge, and Firefox packages are generated.

## Important limitations

- Amazon Ember is not bundled because redistribution/embedding evidence is not established.
- Bundled site selectors retain their actual fixture verification labels; current live-site validation is still required.
- Real-browser E2E, manual accessibility, pinned performance, store signing, and store review remain release blockers.
