# راهنمای نصب RTLX 15.9.1

## Chrome و Microsoft Edge

1. فایل `RTLX-v15.9.1-UNPACKED-INSTALL.zip` را Extract کنید.
2. نسخهٔ قبلی RTLX را در `chrome://extensions` یا `edge://extensions` غیرفعال یا Remove کنید.
3. **Developer mode** را روشن کنید.
4. **Load unpacked** را بزنید.
5. برای Chrome پوشهٔ `LOAD-UNPACKED-CHROMIUM` و برای Edge پوشهٔ `LOAD-UNPACKED-EDGE` را انتخاب کنید.
6. نسخهٔ نمایش‌داده‌شده باید `15.9.1` باشد.
7. صفحهٔ هدف را Refresh کنید یا آن را در Tab جدید باز کنید.

## Firefox Desktop و Android

بسته‌های مربوط به Firefox در تحویل وجود دارند، اما نصب دائمی، امضا و اجرای Android تابع محدودیت‌های Firefox و محیط دستگاه است. نتیجهٔ اجرای واقعی این اهداف فقط زمانی تأیید می‌شود که Artifact نهایی روی مرورگر یا دستگاه مربوط اجرا شود.

## گرفتن گزارش معتبر

- یک پاسخ جدید و ترجیحاً بلند تولید کنید.
- برای آزمایش Streaming یک گزارش حین تولید پاسخ بگیرید.
- گزارش نهایی را بعد از پایان پاسخ و ترجیحاً با `captureReadiness.status: ready` بگیرید.
- روی بخش واقعاً اصلاح‌نشده از ابزار انتخاب بخش استفاده کنید تا `selectedElement` خالی نباشد.

RTLX 15.9.1 پیش از گزارش نهایی تا سقف زمان محدود برای اتمام کارهای داخلی صبر می‌کند، اما کار واقعاً ناتمام را به‌اشتباه `ready` اعلام نمی‌کند.
