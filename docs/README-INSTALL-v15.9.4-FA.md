# راهنمای نصب RTLX 15.9.4

1. بستهٔ `RTLX-v15.9.4-UNPACKED-INSTALL.zip` را استخراج کنید.
2. در Chrome آدرس `chrome://extensions` و در Edge آدرس `edge://extensions` را باز کنید.
3. نسخهٔ قبلی RTLX را حذف یا غیرفعال کنید.
4. Developer mode را فعال کنید.
5. روی **Load unpacked** بزنید.
6. برای Chrome پوشهٔ `LOAD-UNPACKED-CHROMIUM` و برای Edge پوشهٔ `LOAD-UNPACKED-EDGE` را انتخاب کنید.
7. نسخهٔ `15.9.4` را کنترل و صفحهٔ هدف را کامل Reload کنید.

این نسخه تنظیمات ذخیره‌شدهٔ نسخهٔ قبل را حفظ می‌کند.
