# Migration from RTLX 15.9.3 to 15.9.4

No settings or data migration is required.

1. Remove or disable the existing unpacked 15.9.3 extension.
2. Extract `RTLX-v15.9.4-UNPACKED-INSTALL.zip`.
3. Load the directory for the target browser.
4. Reload each already-open target page so the new content runtime starts cleanly.
5. Confirm version `15.9.4` in the extension manager and popup.

Existing site settings remain compatible.
