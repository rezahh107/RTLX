# راهنمای نصب RTLX 15.9.7

## Chrome و Edge

1. فایل ZIP مرورگر را استخراج کنید.
2. صفحهٔ مدیریت افزونه‌ها را باز کنید.
3. Developer mode را فعال کنید.
4. گزینهٔ Load unpacked را بزنید و پوشهٔ استخراج‌شده را انتخاب کنید.
5. صفحهٔ ChatGPT را reload کنید.

## Firefox Desktop

1. فایل ZIP Firefox را استخراج کنید.
2. `about:debugging#/runtime/this-firefox` را باز کنید.
3. گزینهٔ Load Temporary Add-on را انتخاب کنید.
4. فایل `manifest.json` را از پوشهٔ استخراج‌شده انتخاب کنید.
5. صفحهٔ هدف را reload کنید.

## آزمون سریع

- popup نباید روی Checking باقی بماند.
- گزارش bootstrap در صورت خطا باید `schemaVersion: 2.0.0` داشته باشد.
- با نبود تنظیمات اختصاصی سایت، پاسخ context باید همچنان معتبر باشد.
