# راهنمای بستهٔ تحویل RTLX 15.9.3

این نسخه فقط Race تأییدشدهٔ Export گزارش را اصلاح می‌کند. هیچ Profile، Selector، Classifier یا رفتار Streaming تغییر نکرده است.

## فایل‌های اصلی

- `RTLX-v15.9.3-NEXT-DELIVERY.zip`: بستهٔ کامل تحویل؛
- `RTLX-v15.9.3-UNPACKED-INSTALL.zip`: پوشه‌های Load unpacked؛
- `rtlx-*-15.9.3.zip`: بسته‌های چهار هدف مرورگر؛
- `rtlx-v15.9.3-font-sanitized-source.zip`: سورس کامل بدون باینری فونت؛
- `RTLX-v15.9.3-IMPLEMENTATION-REPORT.md/.json`: گزارش تغییرات و اعتبارسنجی؛
- `RTLX-v15.9.3-EVIDENCE.zip`: لاگ‌های اجرای Gateها؛
- `RTLX-v15.9.3-SHA256-MANIFEST.json`: Manifest نهایی تحویل.

## وضعیت Evidence

Gateهای محلی پس از اجرا در گزارش ثبت می‌شوند. تأیید رفتار دقیق تغییر Visibility روی سایت واقعی تا اجرای Artifact نهایی در وضعیت `insufficient_evidence` باقی می‌ماند.
