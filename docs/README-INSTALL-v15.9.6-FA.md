# راهنمای نصب RTLX 15.9.6

1. بستهٔ `RTLX-v15.9.6-UNPACKED-INSTALL.zip` را استخراج کنید.
2. در Chrome نشانی `chrome://extensions` و در Edge نشانی `edge://extensions` را باز کنید.
3. نسخهٔ Unpacked قبلی RTLX را حذف یا غیرفعال کنید.
4. Developer mode را فعال کنید.
5. گزینهٔ **Load unpacked** را بزنید.
6. پوشهٔ `LOAD-UNPACKED-CHROMIUM` یا `LOAD-UNPACKED-EDGE` را انتخاب کنید.
7. نسخهٔ `15.9.6` را کنترل کنید.
8. صفحهٔ DeepSeek را یک‌بار کامل Reload کنید.
9. یک پاسخ فارسی دارای فهرست بسازید و محل Bulletها را کنترل کنید.

این نسخه فقط مالک Direction مربوط به Bullet فهرست‌های تو‌در‌تو را اصلاح می‌کند. برای گزارش نهایی، پس از پایان پردازش صفحه گزارش بگیرید و `captureReadiness.status: ready` را کنترل کنید.
