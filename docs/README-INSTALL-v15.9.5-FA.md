# راهنمای نصب RTLX 15.9.5

1. بستهٔ `RTLX-v15.9.5-UNPACKED-INSTALL.zip` را استخراج کنید.
2. در Chrome نشانی `chrome://extensions` و در Edge نشانی `edge://extensions` را باز کنید.
3. نسخهٔ Unpacked قبلی RTLX را حذف یا غیرفعال کنید.
4. Developer mode را فعال کنید.
5. گزینهٔ **Load unpacked** را بزنید.
6. پوشهٔ `LOAD-UNPACKED-CHROMIUM` یا `LOAD-UNPACKED-EDGE` را انتخاب کنید.
7. نسخهٔ `15.9.5` را کنترل کنید.
8. صفحهٔ هدف را یک‌بار کامل Reload کنید.

برای گزارش نهایی، پس از پایان پردازش صفحه گزارش بگیرید و وضعیت `captureReadiness.status: ready` را کنترل کنید. در Runtime Snapshot نسخهٔ `1.10.0` مقدار `textBlocksProcessingPending` نیز باید صفر باشد.
