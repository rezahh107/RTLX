# راهنمای بستهٔ تحویل RTLX 15.9.5

این بسته شامل موارد زیر است:

- بستهٔ کامل ادامهٔ توسعه؛
- بستهٔ نصب Unpacked برای Chrome و Edge؛
- بسته‌های Chromium، Edge، Firefox Desktop و Firefox Android؛
- سورس کامل Font-sanitized؛
- Patch نسخهٔ 15.9.4 به 15.9.5؛
- گزارش پیاده‌سازی، Evidence و Manifestهای SHA-256.

این نسخه فقط مشکلات تأییدشدهٔ پردازش ناقص Text Block، Readiness نادرست، تزریق تکراری پس از Restart سرویس‌ورکر و پاک‌شدن Selection بر اثر تغییر Runtime را اصلاح می‌کند. ادعای اثربخشی نهایی روی DeepSeek فقط پس از اجرای نسخهٔ 15.9.5 در محیط واقعی و ثبت گزارش جدید مجاز است.
