# راهنمای بستهٔ تحویل RTLX 15.9.1

## هدف نسخه

این نسخه یک Patch هدفمند برای دو نشانهٔ گزارش 15.9.0 است:

- باقی‌ماندن مالکیت و استایل‌های RTLX در DOM بدون ثبت Mutation توسط Runtime جدید؛
- باقی‌ماندن Enumerationهای قطع‌شده در وضعیت Pending و در نتیجه Capture ناتمام.

## فایل‌های اصلی

- `RTLX-v15.9.1-NEXT-DELIVERY.zip`: تحویل کامل؛
- `RTLX-v15.9.1-UNPACKED-INSTALL.zip`: پوشه‌های Load unpacked؛
- `rtlx-*-15.9.1.zip`: بسته‌های چهار هدف مرورگر؛
- `rtlx-v15.9.1-font-sanitized-source.zip`: سورس بدون باینری فونت؛
- `RTLX-v15.9.1-IMPLEMENTATION-REPORT.md/.json`: گزارش اجرا؛
- `RTLX-v15.9.1-EVIDENCE.zip`: لاگ‌ها و شواهد محلی؛
- `RTLX-v15.9.1-SHA256-MANIFEST.json`: هش تحویل نهایی.

## وضعیت ادعا

عبور Gateهای محلی به معنی اثبات بهبود بصری روی سایت واقعی نیست. نتیجهٔ Qwen و Claude برای Artifact نهایی 15.9.1 تا اجرای Capture تمیز و قابل‌مقایسه در وضعیت `insufficient_evidence` باقی می‌ماند.
